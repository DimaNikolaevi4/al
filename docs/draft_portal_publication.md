# ИИ-тьютор: Интеллектуальный помощник для студентов Сальского индустриального техникума

**Проект ГБПОУ РО «Сальский индустриальный техникум»**
**Специальность:** 15.02.14 «Оснащение средствами автоматизации технологических процессов и производств»

---

## Что такое ИИ-тьютор?

ИИ-тьютор — это интеллектуальная система на базе искусственного интеллекта, созданная для помощи студентам в самостоятельном изучении теоретического материала. Система автоматически обрабатывает лекции преподавателей и создаёт структурированные учебные материалы: конспекты, тесты для самопроверки, объяснения сложных терминов и диалоговые ответы на вопросы студентов.

Проект разработан силами преподавателей техникума с использованием современных технологий машинного обучения и больших языковых моделей (LLM). Система полностью развёртывается на собственном сервере техникума — персональные данные студентов не покидают территорию учебного заведения.

---

## Для кого предназначен?

ИИ-тьютор создан для студентов Сальского индустриального техникума, обучающихся по специальности 15.02.14 «Оснащение средствами автоматизации технологических процессов и производств». В рамках пилотного внедрения система будет доступна для групп, изучающих следующие дисциплины:

- **Промышленная автоматизация** — основы АСУ ТП, ПЛК, SCADA-системы, ПИД-регулирование, датчики и исполнительные механизмы
- **Измерительные системы** — метрология, измерительные преобразователи, цифровые измерительные приборы, интеллектуальные датчики

---

## Какую задачу решает?

Сегодня значительная часть аудиторного времени (до 70%) уходит на изложение теоретического материала, тогда как практические навыки — ключевой компонент подготовки специалиста — остаются недостаточно освещёнными. ИИ-тьютор позволяет перераспределить учебный процесс:

- **Студент** изучает теорию самостоятельно дома с помощью ИИ-тьютора, получает конспекты и проверяет свои знания тестами
- **Преподаватель** на занятии сосредотачивается на практической отработке навыков, разборе сложных случаев и индивидуальной работе с отстающими студентами

**Ожидаемый эффект:** повышение среднего балла на 15–20% и высвобождение до 40% аудиторного времени для практических занятий.

## Что умеет ИИ-тьютор?

| Функция | Описание |
|---------|----------|
| **Создание конспектов** | Автоматическая генерация структурированного конспекта по тексту лекции преподавателя. Два уровня сложности: базовый (краткий) и углублённый (с примерами и связями) |
| **Тесты для самопроверки** | Генерация вопросов с вариантами ответов по содержанию лекции. Три уровня сложности: базовый, средний, продвинутый |
| **Объяснение терминов** | Подробное объяснение ключевых понятий дисциплины простым языком с примерами из реальной практики |
| **Диалоговый помощник** | Возможность задать уточняющий вопрос по материалу лекции и получить развёрнутый ответ |
| **Адаптация для студентов с ОВЗ** | Упрощённая версия материалов: короткие предложения, простой язык, отсутствие сложных конструкций |

## Как это работает?

1. Преподаватель загружает лекцию в электронную образовательную среду (Moodle)
2. Студент открывает лекцию и нажимает кнопку «Создать конспект»
3. ИИ-тьютор обрабатывает текст и генерирует структурированный конспект
4. Студент изучает конспект, проходит тесты для самопроверки и может задать вопросы в диалоговом режиме
5. Преподаватель видит статистику использования и может скорректировать учебный процесс

Вся обработка происходит на сервере техникума. Данные студентов не передаются во внешние сервисы.

---

## Безопасность данных

Защита персональных данных студентов — приоритет проекта. Реализованы следующие меры:

- **Локальное развёртывание** — модель и все данные находятся на сервере техникума, доступ в интернет не требуется для обработки
- **Соответствие ФЗ-152** — система спроектирована с учётом требований законодательства о персональных данных
- **Отсутствие облачных сервисов** — в отличие от ChatGPT и аналогов, данные не отправляются на сторонние серверы

## Технологии

Проект построен на современных технологиях с открытым исходным кодом:

- **Языковая модель:** Mistral Small 3.1 (24 млрд параметров) — одна из лучших открытых моделей для работы с русским языком
- **Технология дообучения:** LoRA-адаптеры — позволяют специализировать модель на учебных материалах без переобучения всей модели
- **Программирование:** Python, PyTorch, Transformers
- **Серверная часть:** FastAPI, Celery, Redis
- **Интеграция:** Moodle (через REST API)

Исходный код проекта опубликован под лицензией Apache 2.0 и доступен в репозитории.

## Часто задаваемые вопросы (FAQ)

**Заменит ли ИИ-тьютор преподавателя?**

Нет. ИИ-тьютор — это инструмент помощи, а не замена преподавателя. Система берёт на себя рутинную работу по обработке текстов и генерации конспектов, что позволяет преподавателю уделить больше внимания практике и индивидуальной работе со студентами. Преподаватель полностью контролирует учебный процесс и оценивает знания студентов.

**Что делать, если ИИ-тьютор выдаёт неверный ответ?**

ИИ-тьютор может допускать ошибки — это особенность всех систем искусственного интеллекта на текущем этапе развития. Студентам рекомендуется сверять информацию с учебниками и конспектами преподавателя. Все сгенерированные материалы носят справочный характер. Ошибочные ответы можно сообщить преподавателю для корректировки модели.

**Какие данные видит система?**

Система обрабатывает тексты лекций, загруженные преподавателями в Moodle, и генерирует учебные материалы. Система не имеет доступа к личным данным студентов, не отслеживает личности и не хранит историю диалогов в персонализированном виде. Данные об использовании (количество обращений, дисциплины) агрегируются в общем виде для анализа эффективности проекта.

**Нужен ли студенту специальный навык для работы с тьютором?**

Нет. ИИ-тьютор доступен через привычную среду Moodle. Для работы с системой достаточно базовых навыков работы с компьютером. Интерфейс интуитивно понятен: студент нажимает кнопку и получает готовый конспект или тест.

**Будет ли система работать медленно?**

Время генерации конспекта составляет 3–5 секунд. Для тестов — 5–10 секунд. Система рассчитана на одновременную работу до 25 студентов, что соответствует размеру учебной группы.

**Можно ли использовать ИИ-тьютор на телефоне?**

Да. Доступ через Moodle возможен с любого устройства, включая смартфоны и планшеты. Мобильная версия интерфейса будет адаптирована в рамках пилотного внедрения.
