# Анкета обратной связи: ИИ-тьютор для студентов

> **Задача:** 7.2.3 — Сбор обратной связи (Google Forms / Опросники)
> **Формат:** Шаблон для Google Forms (или аналогичной платформы)
> **Аудитория:** Студенты группы 15.02.14, использующие ИИ-тьютор в рамках пилотного внедрения
> **Время заполнения:** ~5 минут
> **Анонимность:** Полная — имя и группа указываются по желанию

---

## Введение (описание формы)

**Заголовок формы:**
> Обратная связь по использованию ИИ-тьютора

**Описание формы:**
> Уважаемый студент!
>
> Благодарим вас за участие в пилотном внедрении ИИ-тьютора в учебный процесс. Этот опрос поможет нам улучшить качество сервиса и сделать его более полезным для вашей подготовки.
>
> Опрос **полностью анонимен**. Указывать имя и группу — только по вашему желанию. Результаты будут использованы исключительно для улучшения системы.
>
> Время заполнения: **~5 минут**.
>
> Если у вас возникли вопросы по работе ИИ-тьютора, обратитесь в техподдержку через старосту группы или напишите на внутренний портал.

## Рекомендации по проведению опроса

1. **Сроки:** Провести через 2-3 недели после начала использования, затем повторно через 2 месяца.
2. **Канал:** Разослать ссылку через Moodle (уведомление в курсе) и/или Telegram-группу.
3. **Напоминание:** Отправить напоминание через 3 дня после рассылки.
4. **Анализ результатов:**
   - Рассчитать средний балл по каждому из 5 вопросов-шкал (1-5).
   - Рассчитать NPS = % рекомендующих (9-10) − % критиков (0-6).
   - Проанализировать открытые ответы для выявления повторяющихся проблем.
   - Сравнить результаты первого и второго опросов для оценки динамики.
5. **Целевые метрики (из CHECKLIST.md):**
   - Средний балл по дисциплине (влияние ИИ-тьютора).
   - Время на подготовку конспектов (экономия).
   - Удовлетворённость студентов (NPS).
