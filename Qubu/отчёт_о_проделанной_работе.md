## Шаг 4. Расчёт стоимости QLoRA дообучения

**Дата выполнения:** 2026-05-16

### Требования к VRAM для QLoRA 24B

| Компонент | Размер |
|-----------|--------|
| 4-битная базовая модель Mistral Small 24B | ~15 ГБ |
| LoRA-адаптеры (rank=16) | ~0.5 ГБ |
| Состояние оптимизатора | ~1.0 ГБ |
| Градиенты | ~0.25 ГБ |
| Активации (batch=1, context=4096) | ~3–4 ГБ |
| Временные буферы | ~0.5 ГБ |
| **ИТОГО минимум** | **~20–21 ГБ** |
| **Рекомендуемый запас** | **+3–5 ГБ** |
| **ИТОГО рекомендовано** | **~24 ГБ** |

### Финансовый расчёт для Mistral 24B

| Параметр | Значение |
|----------|----------|
| GPU для QLoRA 24B | RTX A5000 (24 ГБ VRAM, 28 коинов/час) |
| Оценка времени обучения (5 эпох) | ~4–6 часов |
| Стоимость обучения | ~112–168 коинов |
| Текущий баланс | 133 коина |
| **Дефицит** | **Недостаточно средств** |

### Финансовый расчёт для Qwen2.5-7B

| Параметр | Значение |
|----------|----------|
| GPU для QLoRA 7B | RTX A4500 (20 ГБ VRAM, 25 коинов/час) |
| Оценка времени обучения (5 эпох) | ~1–2 часа |
| Стоимость обучения | ~25–50 коинов |
| Текущий баланс | 133 коина |
| Остаток после обучения | ~83–108 коинов |

### Вывод

Дообучение 24B модели при текущем балансе (133 коина) **невозможно**. Дообучение 7B модели — финансово осуществимо (25–50 коинов). Однако ограничения платформы (50-минутный таймаут сборки, грантовые лимиты токенов) остаются главной проблемой.
